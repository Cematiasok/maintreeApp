# MaintreeApp

MaintreeApp es una aplicación de asistencia para el mantenimiento industrial, diseñada para gestionar el despiece de maquinaria y sus repuestos en un formato de árbol jerárquico.

## ✨ Características

*   **Visualización en Árbol:** Navega de forma intuitiva por la estructura de despiece de la maquinaria.
*   **Gestión de Repuestos:** Lleva un registro de los repuestos asociados a cada componente.
*   **Búsqueda Rápida:** Encuentra fácilmente máquinas, componentes y repuestos.
*   **Interfaz Web:** Accede a la aplicación desde cualquier dispositivo con un navegador web.

## 🚀 Tecnologías Utilizadas

*   **Backend:** Java
*   **Servlets:** Jakarta Servlet API
*   **JSON:** Google GSON
*   **Servidor de Aplicaciones:** Apache Tomcat 10.x
*   **Gestión de Dependencias:** Maven

## 🏁 Cómo Empezar

Sigue estos pasos para tener una copia del proyecto funcionando en tu máquina local.

### Prerrequisitos

*   Tener instalado JDK 22 o superior.
*   Tener instalado Apache Maven.

### Instalación y Ejecución

1.  **Clona el repositorio:**
    ```sh
    git clone <URL-DEL-REPOSITORIO>
    ```
2.  **Navega al directorio del proyecto:**
    ```sh
    cd maintreeApp
    ```
3.  **Empaqueta la aplicación:**
    ```sh
    mvn package
    ```
4.  **Despliega y ejecuta la aplicación en Tomcat:**
    ```sh
    mvn cargo:run
    ```
5.  Abre tu navegador y ve a `http://localhost:8080/mywebapp/main.html`
